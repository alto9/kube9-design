# kube9 Design System

Vue components and design tokens based on martex-template.

## Quick Start

```bash
npm install @kube9/design
```

```vue
<script setup>
import { Button, Card } from '@kube9/design/components'
import '@kube9/design/styles'
</script>

<template>
  <Card>
    <Button variant="primary">Click me</Button>
  </Card>
</template>
```

## What's Included

- **Vue Components** - Reusable Vue 3 components using martex-template styles
- **Design Tokens** - Colors, typography, spacing extracted from martex-template
- **CSS Foundation** - Complete martex-template CSS (never modified)
