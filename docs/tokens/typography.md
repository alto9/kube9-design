# Typography

Typography tokens from martex-template.

## Font Families

- `font-rubik` - Rubik font family
- `font-jakarta` - Plus Jakarta Sans font family
- `font-inter` - Inter font family

## Usage

```typescript
import { typography } from '@kube9/design/tokens'

typography.fontFamilies.rubik
typography.fontSizes.lg
typography.fontWeights.semibold
```
